# Fifty and Unfiltered — Folder Instructions
### This file lives in your CoWork folder root so CoWork reads it every session

---

## This Folder

This is the working folder for Fifty and Unfiltered, a blog and community for Black women over 50. Everything created in this folder is content for the brand, the website at fiftyandunfiltered.com, or the social media accounts @keepupwithmg and @fiftyunfiltered.

---

## Folder Structure

Always save files to the correct subfolder. Never save anything to the root of this folder.

- **Brand Files** — Brand summary, tone guide, audience description, topics already covered. Read these files at the start of every session before writing anything.
- **Weekly Research** — Monday research summaries and topic lists. One file per week.
- **Blog Drafts** — Completed blog posts and all social media content for each post. One file per topic.
- **Newsletter Drafts** — Monthly newsletter drafts. One file per month.
- **Social Media Content** — Standalone social content not attached to a specific blog post.
- **Archive** — Move anything older than 30 days here. Never delete files.

---

## Before Starting Any Task

1. Read the Brand Summary file in the Brand Files subfolder.
2. Check the Topics Already Covered file in the Brand Files subfolder so content is never repeated.
3. If that file does not exist yet, create it and start tracking topics from this session forward.

---

## File Naming Rules

Use this exact format for every file saved in this folder:

| File Type | Format |
|---|---|
| Blog post + social content | YYYY-MM — Topic Short Title — Blog + Social |
| Weekly research summary | YYYY-MM-DD — Weekly Research Summary |
| Newsletter draft | YYYY-MM — Newsletter Draft |
| Instagram growth report | YYYY-MM — Instagram Growth Opportunities |
| Standalone social content | YYYY-MM-DD — Platform — Topic |

Example: 2026-04 — Menopause and Mental Health — Blog + Social

## Quality Rules

Every piece of content created in this folder must meet these standards before being saved:

- Written in Mary Geter's voice — direct, warm, girlfriend-to-girlfriend
- Never uses: seasoned, delve, streamline, restorative, reclaim, em-dashes, generic empowerment language
- Specific to Black women over 50 — not generic women's content
- Blog posts include at least one real statistic or research reference
- Instagram captions do not exceed the character limits for each option type
- Reel scripts include setting direction and energy direction
- All SEO fields are complete — no blanks

---

## Topics Already Covered

When this file does not exist yet, create it in Brand Files and add every topic written from this session forward. Format:

```
YYYY-MM — Topic Title — Category
```

Check this file before suggesting new topics. Never repeat a topic that has already been covered.

---

## Scheduled Tasks Running From This Folder

**Every Monday (Weekly Research)**
Search the internet for what is currently affecting Black women over 50. Compile a summary and present 10 ranked topic ideas. Save to Weekly Research subfolder.

**First Monday of Each Month (Newsletter)**
Search for 6 to 8 timely topics for the monthly newsletter. Write all snippets. Save draft to Newsletter Drafts subfolder.

**First Monday of Each Month (Instagram Growth)**
Research 5 Black women creators or midlife empowerment accounts currently growing on Instagram. Save report to Social Media Content subfolder.

---

## What Never Goes in This Folder

- Content for Beyond the Hairline (that is a completely separate brand)
- Personal files unrelated to Fifty and Unfiltered
- Duplicate versions of the same file — update the existing file instead
- Files without the correct naming format
